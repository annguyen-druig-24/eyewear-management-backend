==============================
API CART & CART ITEM - FLOW + TEST CASE
==============================

1) Tong quan nhanh
- Base URL: /api/cart
- Controller dang dung CORS: http://localhost:3000
- Cac API deu lam viec theo user dang dang nhap (lay tu SecurityContext)
- Khong truyen userId trong request body/path cho cac API nay
- Kieu loi va message duoc tra qua GlobalExceptionHandler


2) Danh sach API trong CartController
- POST /api/cart/add
	Muc dich: them moi item vao gio, hoac cong don quantity neu item da ton tai

- GET /api/cart/getAllCart
	Muc dich: lay toan bo item cua gio hien tai

- DELETE /api/cart/delete/{cartItemId}
	Muc dich: xoa 1 item khoi gio

- DELETE /api/cart/deleteAllCart
	Muc dich: xoa tat ca item trong gio

- PUT /api/cart/update
	Muc dich: cap nhat quantity cho 1 cartItem


3) Cac quy tac nghiep vu quan trong
3.1 User context
- Moi API deu goi getCurrentUser()
- Neu chua dang nhap/khong hop le -> loi UNAUTHENTICATED
- Neu username khong tim thay trong DB -> loi USER_NOT_EXISTED

3.2 Them item vao gio (POST /add)
- He thong tim cart theo user:
	+ Co roi: dung cart cu
	+ Chua co: tao moi cart
- He thong load entity theo id request:
	+ contactLensId sai -> CONTACT_LENS_NOT_FOUND
	+ frameId sai -> FRAME_NOT_FOUND
	+ lensId sai -> LENS_NOT_FOUND
- Logic tim item trung:
	+ So sanh bo 3 id san pham (contactLens/frame/lens)
	+ So sanh them thong so prescription
	+ Neu cung bo id + cung prescription -> xem la trung item
- Neu trung item:
	+ quantity = quantity cu + quantity moi
	+ itemType duoc tinh lai
- Neu chua trung:
	+ Tao cartItem moi
	+ Gia frame/lens/contact lens lay tu Product.price trong DB
	+ price tong = tong gia cac thanh phan co trong item
- Neu request co thong so prescription (bat ky truong nao co gia tri):
	+ itemType uu tien la PRESCRIPTION
	+ Neu la item moi thi tao CartItemPrescription

3.3 Xac dinh itemType
- Neu co prescription -> PRESCRIPTION (uu tien cao nhat)
- Neu khong co prescription, nhung co bat ky san pham nao cho phep preorder va availableQuantity = 0 -> PREORDER
- Nguoc lai -> ORDER

3.4 Lay gio hang (GET /getAllCart)
- Neu user chua co cart -> tra ve []
- Neu co cart -> map tat ca cartItem sang CartItemResponse

3.5 Xoa 1 item (DELETE /delete/{cartItemId})
- Tim cartItem theo id
- Neu khong ton tai: code hien tai dang nem UNAUTHENTICATED (khong phan biet ro voi NOT_FOUND)
- Neu ton tai nhung khong thuoc user dang dang nhap: UNAUTHENTICATED
- Neu thuoc user: xoa thanh cong

3.6 Cap nhat quantity (PUT /update)
- Validate request:
	+ cartItemId bat buoc
	+ quantity bat buoc, >= 1
- Tim cartItem theo id:
	+ Khong ton tai -> CART_ITEM_NOT_FOUND
- Kiem tra owner cartItem:
	+ Khong phai user hien tai -> UNAUTHENTICATED
- Hop le -> update quantity, luu DB, tra CartItemResponse

3.7 Xoa tat ca gio (DELETE /deleteAllCart)
- Neu co cart: clear danh sach cartItems roi save cart
- Neu chua co cart: khong loi, ket thuc em
